## 📁 How to Prepare Your Data

LSTM-Optuna works best with data in files saved as `.csv`. This is a simple text file where values are separated by commas.

Your data file should:

- Contain one or more columns of numbers.
- Have a clear order from oldest to newest data in rows.
- Avoid empty rows or missing values.

Example of a correct CSV file:

```
Value
23
25
22
24
26
27
```

If your data has multiple columns, you can choose which one to use in the app.

- **Setup file won’t start:** Make sure your antivirus or Windows Defender is not blocking the file. Try right-clicking and selecting “Run as administrator”.
